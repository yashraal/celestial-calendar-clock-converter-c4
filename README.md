# celestial-calendar-clock-converter-c4
A web application for converting pagan gregorian dates to the Scriptural Celestial Calendar. Calculates Appointed Times (Feasts), New Moons, and Shabbats based on the Abib cycle and lunar phases, restoring the ancient timekeeping of Genesis 1:14-16, not solar, lunar or solar-lunar, considering sun, moon and stars, all the three witnesses.
